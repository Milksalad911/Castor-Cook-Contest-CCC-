# 🍔 Castor Cook Contest

Interactieve ticketwebsite voor de **Castor Cook Contest**.  
Bezoekers kunnen een ticket kopen als toeschouwer of deelnemen aan de kookwedstrijd.

Deelnemers maken kans op het winnen van een nieuwe barbecue.

---

## 🎟️ Tickets

- **Toeschouwer** — €2  
- **Deelnemer** — €10
